# 🍎 水果抽签器 - Flask 单页应用（已更新）

一个现代简洁的水果抽签应用，帮你决定明天吃什么水果！

## ✨ 最新更新

### v2.0 更新内容：

1. **✅ 所有水果按钮功能**
   - 点击"所有水果"按钮时才显示20种水果
   - 默认隐藏水果列表，保持界面简洁
   - 平滑滚动到水果展示区

2. **✅ 关于弹窗**
   - 点击"关于"按钮显示弹窗
   - 显示：由 OpenClaw 创建，Python Flask 单网页应用程序
   - 点击关闭按钮或弹窗外部可关闭
   - 自动返回首页

3. **✅ 抽签音乐**
   - 点击"开始抽签"按钮时播放音乐
   - 使用在线音频资源（无需本地文件）
   - 音量适中（30%），不会太吵
   - 抽签动画完成后自动停止

## 🚀 快速开始

### 1. 安装依赖

```bash
pip install flask
```

### 2. 运行应用

```bash
python app.py
```

### 3. 访问应用

打开浏览器访问：**http://127.0.0.1:5001**  或 **http://host:5001**

## 🎨 界面功能

### 导航栏
- **首页**：返回主抽签界面
- **所有水果**：显示20种美味水果网格
- **关于**：显示应用信息弹窗

### 抽签功能
- 点击"开始抽签"按钮
- 播放背景音乐
- 水果快速切换动画
- 显示抽中水果的营养信息
- 提供食用时间和份量建议

### 弹窗
- 渐变紫色背景的标题栏
- 白色卡片内容区
- 关闭按钮（×）
- 点击外部区域也可关闭
- 淡入滑出动画效果

## 🎵 关于音乐

- 使用在线音频：SoundHelix 免费音乐
- 自动播放，无需额外配置
- 音量已调整为30%，不会过于响亮
- 抽签动画完成时自动停止

如需更换音乐，修改 `templates/index.html` 中的音频URL：

```html
<audio id="drawMusic" preload="auto">
    <source src="你的音乐URL" type="audio/mpeg">
</audio>
```
